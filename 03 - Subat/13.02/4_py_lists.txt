## SETS
## Ýndexleri yoktur. Sýralarý yoktur.
## Ekleme ve silme iţlemleri yapýlabilir. Bunun için indexleri deđil SÝLÝNECEK OLAN ELEMAN GEREKLÝDÝR.
## Setlerde YENÝ ELEMANLAR EKLENEBÝLÝR. AMA BENZERSÝZ OLMALARI ZORUNLUDUR.
## Boţ bir set oluţturmak içn {} kullanlamaz Bu boţ dicitonary oluţturacaktýr.
## Eđer boţ set oluţturmak istiyorsak o zaman set() constructor kullanýlabilir.
## Stringleri set ile kullanabiliriz. Stringler ile kullanýldýđý zaman BENZERSÝZ OLAN karakterleri ekleyecektir.
## Bunu yapabilmek için set() metodunu kullanmamýz ţarttýr.
## setlerde remove metodu ve discard metodunun temel farký remove bulamaz ise error verir, discard vermez.

string_set = set("çekoslavakyalýlaţtýramadýklarýmýzdanmýsýnýnýz")
basket = {'apple', 'orange', 'apple', 'pear', 'orange', 'banana', 'banan','app' , 'different'}

# pop() metodu setler ile kullanýldýđý zaman RASTGELE bir elemaný silecektir ve bu elemaný geriye deđer olarak döndürecektir.

# set.difference() method. Setlerin farkýný alýr. Sadece soldaki setin deđerlerini alacaktýr.
# Immutable bir metottur bu yüzden yeni SET döner.
# difference alýnýrken birden fazla set kullanýlabilir. Diđer setlere göre benzersiz olan elemanlarý bulur.
# Her zaman soldaki set esas alýnýr.

# set.difference_update() Mutable özellik taţýr. Hangi set iţlemi çađýrmýţ ise onun elemanlarý artýk yeni

## Frozenset
## Mutable olan SET nesnelerini immutable yapmak için kullanýlýr.
## Yani artýk yeni deđerler eklenemeyecek. zaten deđerleri deđiţtirilemiyordu. sadece yenileri de eklenemez oldu.

fruits = {"apple", "banana", "orange", "strawberry",  "mangoo"}
fruits_frozen = frozenset(["apple", "banana", "orange", "strawberryY" , "mangoO"])
fruits2_frozen = frozenset(("apple", "banana", "orange"))
print(fruits_frozen) 


## Intersection
## Belirlenen bütün setlerin kesiţimlerini alýr. Yeni set döner.
## ayný difference'da da oludđu gibi burada da update versiyonu vardýr.
## Tamamen ayný mantýkta çalýţýr. FROZENSET IMMUTABLE OLDUGU ICIN UPDATE OLAMAZ.

new_tuple = fruits_frozen.intersection(fruits2_frozen)
fruits.intersection_update(fruits2_frozen)

print(f"Fruits intersection update ---->>> {fruits}")
print(new_tuple)
print(fruits_frozen)

## kesiţimleri olmayan setleri belirlemek için isdisjoint() metodu kullanýlabilir.
## True vey False deđerler dönecektir. aldýđý parametreler arasýnda diđer set vardýr.
is_disjoint = fruits.isdisjoint(fruits_frozen)
print(is_disjoint)

## issubset() karţýlaţtýrýlan setin diđer set içerisinde olup olmadýđýný kontrol etmek için kullanýlýr.
## tamamen ayný ise de True döner.
is_subset = fruits.issubset(fruits_frozen);
print(is_subset)

## issuperset argüman olarak gönderilen set içerisindeki bütün elemanlarý
## soldaki set içeriisinde var ise true döner.
is_superset = fruits_frozen.issuperset(fruits);
print(is_superset)


## symmetric difference
## Her iki set içerisinde de ortak olan elemanlarýn çýkartýlýp sadece unique olanlarla yeni set döner.
## symmetric_difference_update() metodu mutable olarak iţlem yapar.
simetrik_tuple = fruits.symmetric_difference(fruits_frozen)
print(simetrik_tuple)

## UNION
## Sete yeni seti ekler. Bütün elemanlarý ekler AMA DUPLÝCATE ÝZÝN VERMEZ.
## Yeni set döner ORTAKLAR SADECE TEK SEFER GECER BÝRDEN FAZLA EKLENEBÝLÝR.
new_set = fruits_frozen.union(fruits)
print(new_set)
