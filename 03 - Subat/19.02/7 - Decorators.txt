## Decorators
## fonksiyonlarý modifiye etmek için kullanýlabilir.
## Argüman olarak fonksiyonlarý alýrlar. bu fonksiyonu kendi içerisinde çađýrmasý yanýnda
## yeni özellikleri de ekler.

## Function decorator
def simple_decorator(func):
    def wrapper():
        print(">>> Starting function")
        func()
        print(">>> Function finished")
    return wrapper

@simple_decorator
def greet():
    print("Hello, World!")
    
greet()

def decor(other_func):
  def wrapper(parametre):
    print("once")
    other_func(parametre)
    print("sonra")
  return wrapper
  
def func(parametre):
  print(parametre)
  
func = decor(func)
func("Deneme")

## method Decorators
## nesnelerin metotlarýna uygulanýr.
## Deđiţtirilecek olan metodun aldýđý parametre wrapper fonksiyona parametre olarak gönderilmesi lazým.
def method_decorator(func):
    def wrapper(self):
        print("Before method execution")
        res = func(self)
        print("After method execution")
        return res
    return wrapper

class MyClass:
    @method_decorator
    def say_hello(self):
        print("Hello!")
        
obj = MyClass()
obj.say_hello()


## CLASS Decorators
## mevcut sýnýflara yeni özellikleri eklemek için kullanýlýr.
## parametre olarak sýnýf ismini alýr.

def fun(cls):
    cls.class_name = cls.__name__
    return cls

@fun
class Person:
    pass
print(Person.class_name)
